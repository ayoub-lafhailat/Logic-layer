# Canon PDF Generator

## Overview

This project was developed as part of a team-based software engineering project for Canon.

The goal of the application was to automatically generate randomized PDF files and images that complied with predefined testing criteria. These generated files could then be automatically sent to Canon printers for testing and validation purposes.

The project focused on creating a scalable and structured software solution using layered architecture principles and modern software engineering practices.

Out of multiple student teams working on the assignment, Canon selected and used our final solution.

# Project Goal

Canon required a system capable of automatically generating large amounts of randomized PDF test files that met specific technical criteria.

These files were intended to support automated printer testing workflows and reduce the need for manual test file creation.

The project focused on:
- automation
- scalability
- structured software design
- maintainability

---

# Architecture

The application was designed using a layered architecture approach to improve maintainability and separation of responsibilities.

The architecture included:

- Presentation Layer
- Business Logic Layer
- Data/Service Layer
- PDF Generation Components

This structure allowed the project to remain modular and easier to extend or maintain.
